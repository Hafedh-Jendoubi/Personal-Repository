Si vous ne souhaitez pas que les valeurs soient dupliquées dans la base de
données, Set doit être votre premier choix, car toutes ses classes
n’autorisent pas les doublons.

Si des opérations de recherche fréquentes basées sur les valeurs d’index
sont nécessaires, alors List (ArrayList) constitue un meilleur choix.

S’il est nécessaire de maintenir l’ordre d’insertion, List est également
préférée.

Si l’exigence est d’avoir un mappage clés/valeurs, alors Map est votre
meilleur choix.
