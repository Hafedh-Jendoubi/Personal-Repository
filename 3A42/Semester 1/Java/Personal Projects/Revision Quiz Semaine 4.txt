Notes about Java:

CHAPITRE 1

Java:

Java est un langage POO, compilé et interprété. 
.java -> .class (Compliation)
JVM (Java Virtual Machine) -> .class files (Interpretation)

----------

Classe en java:

Les données d'une classe (Attributs) Partie Statique
Le code les manipulant (Methodes) Partie Dynamique

----------

Type de donnée en java:

Primitifs: contiennent la valeur réelle de la var en mémoire. (int/float/double)
Référence: contient l'adresse mémoire où l'information est réellement stockée (classe)

----------

Outils de développement:

IDE (Integrated Development Environment): IDE deja na3rfou comme IntelliJ/CodeBlocks etc..
JDK (Java Developement Kit): un ensemble d'outils de développement pour comprendre le compilateur
JRE (Java Runtime Environment): un ensemble d'outils nécessaires pour exécuter les apps java

------------------------------

CHAPITRE 2:

Constructeur d'une classe:

Le rôle d’un constructeur est d'initialiser les données membres de la classe.
Le mot-clé <<this>> permet de désigner l'objet courant

----------

Instancier un objet:

Candidate c1 = new Candidate();

----------

Méthodes:

Une méthode est une fonction faisant partie d'une classe.


------------------------------

CHAPITRE 3:

Création et remplissage du tableau:

//Création
int[] tab = new int[3]; //exemple
//Remplissage
String[] animaux = {"chat", "poisson", "oiseau"};
Ou
String[] animaux = new String[3];
animaux[0] = new String("Chat"); //exemple remplissage 1
animaux[1] = "Poisson"; //exemple remplissage 2

----------

Manipulation des tableaux:

Boucle for or this new one:

for(int i: tab){
	Sout(i);
}

----------

Le mot clé Static et Final:

Une variable statique ou une méthode statique est partagée par toutes les instances de la classe.
Les variables statiques sont accessibles dans tous les types de méthodes : statiques ou non statiques.
